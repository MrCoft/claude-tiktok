Claude Code plays a microwave ding sound when it needs you — then a short TikTok-voice recap of what it just said.

## Requirements

- Windows
- **Anthropic API key** with prepaid credits (separate from your Claude Code subscription; ~$0.0002 per summary via Haiku 4.5 — $5 buys roughly 25,000 summaries)

## Install

```text
/plugin marketplace add <source>
/plugin install claude-tiktok@claude-tiktok-local
```
